String

String

Object

Key/Value pairs of tags set on the Building Block Definition. Keep in mind, that values are an array. Also see our general section about [tags]

Object

All fields in this section describe the Building Block Definition.

String

The display name of the Building Block Definition as it is shown in meshPanel.

spec.description

String

spec.supportUrl

String

The URL pointing to support of the Building Block Definition. (optional)

spec.documentationUrl

String

The URL pointing to documentation of the Building Block Definition. (optional)

spec.supportedPlatforms

Array

List of the platforms this Building Block Definition supports

spec.runTransparency

Boolean

Specifies the Building Block Definition run control. When set to true, both platform teams and workspace users can view detailed run logs and re-run building blocks. When set to false (default), only platform teams have this access.

spec.useInLandingZonesOnly

Boolean

Indicated whether this Building Block Definitions should be used on Landing Zones only.

spec.notificationSubscriberUsernames

Array

A list of usernames from the users that should be notified as subscribers for this Building Block Definition.

spec.symbol

String

The symbol of the Building Block Definition as it is shown in meshPanel.

spec.targetType

String

The type of Building Block Definition. (One of [TENANT_LEVEL, WORKSPACE_LEVEL])

status

Object

All fields in this section describe the current status of the Building Block Definition.

status.versions[]

Array

List of meshBuildingBlockDefinitionVersions, which exist for the Building Block Definition.

status.latestVersion

Number

The number of the latest version based on this Building Block Definition including a possible draft version.

status.latestReleasedVersion

Number

The number of the latest version based on this Building Block Definition not including a possible draft version.

status.versions[].versionNumber

Number

The number of the listed version.

status.versions[].versionUuid

String

The uuid of the listed version

status.versions[].state

String

The state of the listed version. (One of [DRAFT, RELEASED])
